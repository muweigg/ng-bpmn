# ng bpmn demo
ng bpmn demo.

### Quick start
```bash
# Clone my repo
git clone https://github.com/muweigg/ng-bpmn.git

# Change directory to ng-bpmn
cd ng-bpmn

# Install dependencies
npm install
or
yarn install

# Start dev server at http://localhost:4444
npm run serve
or
yarn serve

# Enable hot module replacement
npm run serve:hot
or
yarn serve:hot

# Build for production (uses aot compiler)
npm run build:aot
or
yarn build:aot
```
